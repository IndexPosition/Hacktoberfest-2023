You are given a value x. You need to print elements of Pascal's triangle corresponding to xth row.

### Input Format

First line of input contains a single integer x.

### Constraints

1 <= x <= 20

### Output Format

Print space separated elements of xth row of triangle.

```
Example: 1

Input:
3
Output:
1 3 3 1

Example: 2

Input:
4
Output:
1 4 6 4 1

```
